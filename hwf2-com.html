Resideo Fire Communicator HWF2-COM

    Key Features

    • LTE and IP connection tested every day
    • Use the 7AH battery (not included) to provide 24 hour backup capability
    • Three selectable reporting paths: LTE cellular only, IP only, or IP Primary with LTE cellular backup
    • Connects directly to the primary and secondary telephone ports of a DACT
    • Plan choices range from 5 minute, 60 minute, 6 hour, and 24 hour supervision intervals
    • Operates over the following communication protocols: LTE, HSPA (4G) HSPA (3G)

    • Data transmits over standard Contact ID protocol but is secured with the industry's advanced encryption standard (AES 256 bit)
    • Supports both dynamic (DHCP) or public and private Static IP addressing
    • Reliable connection: IP and cellular connection tested every day

Enhance your commercial fire alarm system with the Resideo HWF2-COM Series LTE / IP fire communicator, designed for seamless integration with any FACP. Featuring Contact ID reporting and dual path connectivity options, this communicator ensures reliable transmission of signals to the AlarmNet network control center. Choose from LTE cellular only, IP only, or IP primary with LTE cellular backup reporting paths for added flexibility. Rest easy knowing that your system is monitored around the clock with highly secure, encrypted data packets delivered via Internet connection or LTE cellular network. Simplify installation and programming with the user-friendly handheld 7720P Programming tool included.
