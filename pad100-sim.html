Potter Addressable Input Module PAD100-SIM

    Key Features

    • 1 Class B monitoring input
    • SLC Class A, Class X and Class B
    • Mounts in a standard 4" or double gang box
    • Wiring terminals accessible when mounted in box
    • All wiring terminals accept 22 to 12AWG

The Potter Electric Signal, LLC PAD100-SIM Addressable Input Module is a versatile and reliable solution for monitoring Class B circuits. Designed to easily mount on a 4" square or double gang box, this module features one red LED that flashes to indicate its status when polled by the control panel. When activated, the LED flashes rapidly, providing clear visual feedback. Compatible with Potter's IPA and AFC/ARC series addressable fire alarm control panels, the PAD100-SIM is an essential interface module for monitoring dry contact devices like sprinkler waterflow, valve tamper switches, and conventional pull stations. Trust the PAD100-SIM for seamless integration and enhanced safety in your fire protection system.
